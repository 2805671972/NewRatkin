랫킨 왕국
랫킨 왕국은 당신의 세력과 우호적일 것입니다. 적어도 당신이 나쁜짓을 하지 않는다면 말이죠.

스토리텔러 로티 마일든
로티 마일든은 랜덤 이벤트에 기반을 둔 스토리 텔러입니다.
- 작은 사건사고에 대한 이벤트 횟수가 증가합니다
- 큰 사건사고에 대한 이벤트 횟수가 조금 줄어듭니다.
- 완전한 랜덤에 의해 이벤트가 동시에 겹쳐 정착지가 박살나는 것을 방지합니다.
